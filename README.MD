# ngx-form-object

Ng-form-object is an abstration on top of Angular's FormBuilder.

## Development

To generate all `*.js`, `*.d.ts` and `*.metadata.json` files:

```bash
$ npm run build
```

To lint all `*.ts` files:

```bash
$ npm run lint
```
